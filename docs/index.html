Usage

Getting data here

  1. Run the thermal server on the LAN as usual (`cd sauron/src && python3 http_thermal_server.py`).
  2. On a gateway with LAN + internet, run `python3 sauron/src/github_pages_pusher.py` (from repo root) to pull frames/status and write to docs/data/.
  3. Commit and push the updated files to GitHub. With Pages set to the docs/ folder, this site will refresh automatically.

Tip: add a cron job or systemd service to run the pusher every few seconds/minutes. The viewer polls for updates; query params bust the cache.
